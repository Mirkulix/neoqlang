🔬 WISSENSCHAFTLICHER HINTERGRUND
───────────────────────────────────────────────────────────────────────

IGQK basiert auf:
  - Information Geometry (Fisher-Information Metrik)
  - Quantum Mechanics (Quantum Gradient Flow)
  - Riemannian Geometry (Natürliche Gradienten)
  - Rate-Distortion Theory (Optimale Kompression)

Mathematische Formel:
  dρ/dt = -i[H, ρ] - γ{∇L, ρ}

Wo:
  ρ  = Quantum State (Ihre Modell-Gewichte)
  H  = Hamiltonian (Energie-Operator)
  ∇L = Gradient (Lernrichtung)
  γ  = Dämpfung

Resultat: Bessere Minima + Stärkere Kompression!
